# Quant Alternative Data Alpha Mining

This project explores the use of alternative data sources 
(job postings and social discussion) to generate predictive 
signals for short-term equity returns.

## Research Goal
Investigate whether attention spikes and hiring momentum 
contain predictive information for 5-day forward returns.

## Data Sources (V1)
- Company job postings
- Reddit discussion volume

## Methodology
- Leakage-safe feature engineering
- Walk-forward validation
- Transaction cost-aware backtesting

This is a research project for educational purposes only.
